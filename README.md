# 📺 Netflix Data Analysis Dashboard

An interactive Streamlit web application that performs data analysis on the Netflix Movies and TV Shows dataset using Pandas, NumPy, and Matplotlib.

## 🚀 Features

- Upload Netflix CSV dataset
- Dynamic filtering using sidebar
- Numeric range selection
- Statistical analysis using NumPy
- Interactive charts (Line, Bar)
- Histogram visualization using Matplotlib

## 🛠 Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- Matplotlib
